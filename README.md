This project is an AI-driven multilingual chatbot web application designed to assist users in exploring Indian government scholarships and welfare schemes. It provides personalized recommendations based on user input such as category, income, education level, and location.

🚀 Key Features
🔍 Scholarship & scheme discovery
🤖 AI-based chatbot interaction
🌐 Multilingual support (Hindi, English, etc.)
📊 Eligibility checking system
🧾 Document and application guidance
⚡ User-friendly web interface
